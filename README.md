# 函館卍會

## プロダクト概要
 - クリスマスツリーの飾り付け
 - みんなを笑顔に
 - GPSを使って、特定の場所に近づくとツリーに装飾できる。
 - たくさんの装飾をしてツリーを華やかにしよう。
