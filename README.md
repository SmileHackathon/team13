# 函館卍會

## プロダクト概要
 - クリスマスツリーの飾り付けアプリ（ジュリーンジュリーンズ）
 - みんなを笑顔に。
 - GPSを使って、特定の場所に近づくとツリーに装飾できる。
 - 飾り付けしたツリーの数をユーザー間で共有できる。
 - たくさんの装飾をしてツリーを華やかにしよう。

## 開発技術
 - C#
 - Unity
 - github
 - GPS
 - firebase
